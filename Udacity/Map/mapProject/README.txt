This is the project for the Udacity Neighborhood Map
The app contains a list of 40 bars around Denver CO.
The data is fetched from foursquare asynchronously using the explore command.
This is app is built using the KnockoutJS organizational framework.

featurewise - 
There is a list of all bars along the left hand side of the map.
This can be opened or closed using the hamburger icon to the right of the list.

The list can be filtered using the input box at the top of the list, the list is dynamically filtered as you type.

further updates include adding the ability to dynamically get new data from foursquare as the map is moved.

Open index.html in the browser to launch the map app.
