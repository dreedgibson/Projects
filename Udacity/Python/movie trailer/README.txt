In order to view the movie website open and run the entertainment_center.py file.
To add further movies to the movie website:
1) create a new movie object by using the following code:
var = movie.media(title, description, poster image url, trailer url, rating)
2) note that rating can only be one of: G, PG, PG-13, and R
3) add the new movie to the movies list currently on line 62.
